## 🐛 如果遇到问题

### 问题 1：关系线上看不到文字

**可能原因：**
- 浏览器缓存
- CSS 没有加载

**解决方法：**
1. 按 Ctrl+F5 强制刷新
2. 清除浏览器缓存
3. 检查开发者工具 Console 是否有错误

### 问题 2：节点位置不对

**可能原因：**
- 旧的位置数据缓存

**解决方法：**
1. 关闭人物关系窗口
2. 刷新页面（F5）
3. 重新打开人物关系窗口

### 问题 3：样式看起来不对

**可能原因：**
- CSS 文件没有更新

**解决方法：**
1. 检查 Network 标签，确认 CSS 文件加载
2. 强制刷新（Ctrl+F5）
3. 重启服务器
