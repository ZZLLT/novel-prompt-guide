## 第1步：验证配置

### 在浏览器中打开

**前端地址**: http://127.0.0.1:5890

### 检查API设置窗口

1. 点击顶部的 "⚙️ API设置" 或 "🔑 API设置"
2. 确认以下设置：
   ```
   ✅ 端点: https://api.deepseek.com/v1
   ✅ 模型: deepseek-chat
   ✅ API密钥: sk-********* (已填写)
   ✅ 启用即时API: ✓ (勾选)
   ```
3. 确认底部状态显示："已保存密钥，当前走队列" 或 "已启用即时API"

## 🐛 如果遇到问题

### 问题1：AI面板打不开
**解决：**
- 刷新页面（F5）
- 检查浏览器控制台（F12）是否有错误
- 确认快捷键是否正确（Ctrl+K）

### 问题2：AI无响应
**检查：**
1. 打开浏览器控制台（F12）
2. 切换到Network标签
3. 发送消息时观察请求
4. 如果看到红色错误，检查：
   - API密钥是否正确
   - 网络是否正常
   - 后端服务器是否运行

### 问题3：报错"API key not configured"
**解决：**
1. 重新打开API设置
2. 重新填写API密钥
3. 确保勾选"启用即时API"
4. 点击"保存配置"
5. 刷新页面

### 问题4：报错401 Unauthorized
**原因：** API密钥无效
**解决：**
1. 访问 https://platform.deepseek.com
2. 检查API密钥是否正确
3. 检查账户是否有余额
4. 重新复制密钥并配置

### 问题5：报错403 或网络超时
**原因：** 网络问题
**解决：**
1. DeepSeek API一般不需要代理
2. 检查防火墙设置
3. 尝试ping api.deepseek.com
