## 🤖 测试AI功能（需要API密钥）

### 方案A：使用OpenAI API

#### 1. 获取API密钥
访问：https://platform.openai.com/api-keys

#### 2. 配置方式1：环境变量
```bash
# Windows CMD
set OPENAI_API_KEY=sk-your_key_here

# Windows PowerShell
$env:OPENAI_API_KEY="sk-your_key_here"

# 重启服务器
cd server
python main.py
```

#### 3. 配置方式2：Web界面
1. 访问 http://127.0.0.1:5890
2. 点击顶部"API设置"
3. 填写API密钥
4. 保存

#### 4. 测试AI对话
1. 按 Cmd+K 打开AI面板
2. 输入："创建一个冷酷的反派角色"
3. 观察流式响应

#### 5. 测试AI生成
在AI对话中输入：
- "生成主角登场的开场戏"
- "帮我规划一条复仇剧情线"
- "把这段话扩写到500字"

## 🐛 常见问题

### 问题1：AI面板打不开
**解决：**
- 检查快捷键：Cmd+K (Mac) 或 Ctrl+K (Win)
- 或点击顶部"AI 对话"按钮

### 问题2：AI无响应
**解决：**
- 检查后端是否运行：访问 http://127.0.0.1:8000/api/health
- 检查API密钥是否配置
- 查看浏览器控制台错误

### 问题3：生成的内容是空的
**原因：**
- AI未启用（api_enabled: false）
**解决：**
- 配置API密钥
- 确保后端日志显示"AI客户端已初始化"

### 问题4：网络错误
**原因：**
- OpenAI API被墙
**解决：**
- 使用代理
- 或使用DeepSeek等国内API
- 或使用本地模型（Ollama）
