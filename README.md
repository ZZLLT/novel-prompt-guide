# Novel Prompt Guide

长篇小说 AI 写作工作台。它把初设、世界观、人物关系、剧情大纲、章节规划、提示词、WPS 文档和多模型协作放在同一个桌面界面里，适合写小说文档，不是视频或角色声音工具。

## 直接启动

推荐入口：

```bat
start.bat
```

也可以双击：

- `打开项目.bat`
- `启动.bat`

启动后浏览器会打开：

```text
http://127.0.0.1:5890/
```

保持命令窗口开启。关闭窗口会停止本项目。

## 开发版

```bat
dev.bat
```

开发版会启动 Vite，并打开：

```text
http://127.0.0.1:5173/
```

## 核心功能

- 小说初设引导：先填写书名、类型、文风、主角、核心冲突、特殊规则和第一卷目标。
- 项目设定：查看 WPS、LLM 状态和故事阶段。
- 章节工作台：生成提示词、读写 WPS、打开各类功能窗口。
- 窗口化工作流：预算、模式、蓝图、文档、流水线、审计、人物、上下文、伏笔、规划都可以作为独立窗口打开。
- 叙事引擎窗口：把叙事状态快照、章级摘要链、故事线 DAG、伏笔注册表和质量治理闭环放进一个可协作窗口。
- 工作组快捷打开：规划工作组、人物线工作组、成稿工作组会一次打开相关窗口，但不会自动调用 AI。
- 人物关系窗口：支持缩放、拖动、关系线建议提交，并把建议钉到图中。
- AI 协作台：可以手动向 AI 提问，获取提示词和下一步优化建议。
- 多模型协作：可为规划、写作、审校、助手分别指定不同模型。
- Token 控制：短上下文、请求缓存、生成模式降级、即时 API token 守卫。

## API 设置

打开项目后点击右上角 `API 设置`。

可配置：

- API 地址，兼容 OpenAI Chat Completions。
- 默认模型。
- API Key。
- 规划模型、写作模型、审校模型、助手模型。
- 温度和最大输出。

模型列表只会在点击 `获取模型` 时拉取。默认走队列，只有勾选 `启用即时 API` 后才会直接调用外部模型。

配置保存在本项目的 `llm_config.json`。前端不会回显已保存的密钥。
