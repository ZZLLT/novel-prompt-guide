### 2. 提示词模板优化 ✅

**文件：** `server/ai_service.py`（优化3个核心模板）

#### 2.1 角色生成优化

**改进前 vs 改进后：**

| 维度 | 改进前 | 改进后 |
|------|--------|--------|
| 思考深度 | 基础描述 | 6大维度深度思考 |
| 背景故事 | 200字 | 300字+转折点 |
| 外貌描述 | 简单描述 | 50-100字生动描写 |
| 返回字段 | 9个 | 14个（+desire/conflict/habits/speech/arc） |
| 性格刻画 | 3个特征 | 4个特征+内心冲突 |

**新增6大思考维度：**
1. 核心冲突 - 内心最大矛盾
2. 塑造因素 - 什么经历塑造了性格
3. 深层渴望 - 最深渴望和遗憾
4. 行为模式 - 如何权衡选择
5. 语言特点 - 说话方式和习惯
6. 成长空间 - 转变潜力

**新增字段：**
- `desire` - 最深渴望
- `conflict` - 内心冲突
- `habits` - 行为习惯
- `speech` - 语言风格
- `arc` - 成长轨迹

---

#### 2.2 场景生成优化

**改进前 vs 改进后：**

| 维度 | 改进前 | 改进后 |
|------|--------|--------|
| 字数要求 | 1500-2000字 | 1800-2500字 |
| 创作指导 | 5点要求 | 8大要点+4段结构 |
| 结构化 | 无明确结构 | 开场-发展-高潮-结尾 |
| 返回字段 | 9个 | 12个（+hooks/conflicts/emotional_beats） |
| 对话占比 | 未指定 | 明确30-40% |
| 情绪转折 | 未指定 | 至少2-3个 |

**8大创作要点：**
1. 开场钩子 - 前50字抓住注意力
2. 环境渲染 - 五感细节营造氛围
3. 冲突设计 - 每个场景都有冲突
4. 对话真实 - 符合性格，有潜台词
5. 节奏把控 - 快慢结合张弛有度
6. 情绪层次 - 起伏和转折
7. 伏笔暗示 - 为后续埋设
8. 结尾悬念 - 引导继续阅读

**4段式结构：**
```
开场（200字）：环境+钩子，立即抓住注意力
发展（800字）：冲突展开，对话推进，细节渲染
高潮（600字）：冲突爆发，情绪顶点，关键转折
结尾（400字）：暂时解决，留下悬念或新问题
```

**新增字段：**
- `hooks` - 埋下的伏笔
- `conflicts` - 冲突点列表
- `emotional_beats` - 情绪节奏

---

#### 2.3 剧情线生成优化

**改进前 vs 改进后：**

| 维度 | 改进前 | 改进后 |
|------|--------|--------|
| 理论基础 | 无 | 三幕结构理论 |
| 节点数量 | 示例2个 | 标准8个 |
| 节点描述 | 简单描述 | 100字详细描述 |
| 返回字段 | 7个 | 10个（+theme/stakes/resolution/act） |
| 指导维度 | 5点要求 | 6大关键要素 |

**三幕结构：**
- 第一幕（建立）：介绍人物、建立世界观、提出冲突
- 第二幕（对抗）：冲突升级、挫折与成长、中点转折
- 第三幕（解决）：高潮对决、问题解决、余韵回响

**6大关键要素：**
1. 核心冲突 - 明确主要矛盾
2. 角色动机 - 深层驱动原因
3. 升级阶梯 - 逐步升级紧逼
4. 意外转折 - 2-3个合理转折
5. 情感节奏 - 有希望有绝望
6. 主题深化 - 价值思考

**标准8个节点：**
1. 起点：建立冲突（第一幕）
2. 第一次挫折（第一幕）
3. 承诺点：不归路（第二幕）
4. 中点转折（第二幕）
5. 最黑暗时刻（第二幕）
6. 顿悟突破（第三幕）
7. 最终对决（第三幕）
8. 新世界（第三幕）

**新增字段：**
- `theme` - 探讨的主题
- `stakes` - 失败的代价
- `resolution` - 预期结局
- `act` - 所属幕次（每个节点）
