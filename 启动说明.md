# 启动说明

推荐入口：

1. 双击 `start.bat`
2. 浏览器会自动打开 `http://127.0.0.1:5890/`
3. 保持命令窗口不要关闭，关闭窗口会停止项目

兼容入口：

- `打开项目.bat`：会转到 `start.bat`
- `启动.bat`：旧入口，会转到 `打开项目.bat`
- `dev.bat`：开发版入口，启动 `http://127.0.0.1:5173/`
- `启动开发版.bat`：会转到 `dev.bat`

API 切换：

1. 打开项目后点击右上角 `API 设置`
2. 填写兼容 OpenAI Chat Completions 的 API 地址、模型和 API Key
3. 点击 `保存 API`

配置会保存在本项目的 `llm_config.json`，前端不会显示已保存的密钥。

首次写小说：

打开项目后会先弹出 `小说初设引导`。先填写书名、类型、文风、一句话设定、主角、核心冲突、特殊规则和第一卷目标。保存后，这些内容会自动填到工作台对应字段里。
