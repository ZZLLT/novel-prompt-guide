### 3️⃣ 居中显示优化 ✅

#### 布局方式
```css
.relationship-window-layer {
  display: flex;
  align-items: center;
  justify-content: center;
}
```

**Before:**

- pointer-events: none（点击穿透）
- 不居中

**After:**
- Flexbox居中
- pointer-events: auto（可交互）
- 完美居中

**效果：**
- ✅ 自动水平垂直居中
- ✅ 适应各种屏幕尺寸
- ✅ 不需要手动定位
