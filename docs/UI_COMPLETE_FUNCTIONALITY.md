## 📋 功能窗口详细说明

### 预算控制窗口
**用途**：管理 Token 使用，避免超支

**功能点**：
- 实时显示已使用 Token
- 设置预算上限
- 设置警告阈值
- 查看消耗趋势（占位）

**适用场景**：
- 开始新任务前设置预算
- 监控 Token 使用情况
- 接近上限时收到警告

### 生成模式窗口
**用途**：选择 AI 生成策略

**3 种模式**：
1. **精确模式** - 严格遵循，重要章节
2. **平衡模式** - 质量与创意兼顾（推荐）
3. **创意模式** - 更多发挥空间

**适用场景**：
- 日常写作用平衡模式
- 关键章节用精确模式
- 头脑风暴用创意模式

### 文档工作台窗口
**用途**：与 WPS 同步文档

**功能点**：
- 显示当前文档名
- 最后同步时间
- 文档字数统计
- 读取/写入操作

**适用场景**：
- 从 WPS 导入已有内容
- 将生成内容导出到 WPS
- 同步本地与云端
