### 2. 人物关系图右侧面板优化

**问题：** 右侧侧边栏样式不够精致，缺少视觉层次

**优化内容：**

#### ✅ 面板整体升级
```css
宽度：280px → 320px（更宽松）
背景：纯色 → 米色背景
边框：1px → 2px（更明显）
内边距：添加统一间距
子元素间距：gap: var(--sp-4)
```

#### ✅ RelationshipLog优化
```css
背景：白色卡片
边框：2px solid
圆角：16px
阴影：柔和阴影
头部：分隔线+文字样式
最大高度：300px可滚动
```

#### ✅ 关系卡片优化
```css
边框：1px → 2px
背景：渐变背景（根据关系类型）
Hover效果：左移2px + 阴影
```

**关系类型渐变背景：**
- 盟友(alliance)：柔和绿渐变
- 紧张(tension)：柔和桃渐变  
- 对立(rivalry)：柔和粉渐变
- 师徒(mentor)：淡紫渐变
- 恋爱(romance)：粉色渐变

#### ✅ 预设面板优化
```css
背景：白色卡片
边框：2px
圆角：16px
头部：分隔线
Select：加粗边框+hover效果+focus外发光
Summary：米色背景+圆角
Chip按钮：加大+hover上浮+颜色变化
```

**效果：**
- ✅ 侧边栏视觉层次清晰
- ✅ 每个面板独立卡片化
- ✅ 交互反馈丰富
- ✅ 关系类型一目了然

### 3. 文字和UI贴合度优化

**问题：** 文字和UI元素间距、字号不够协调

**优化内容：**

#### ✅ 全局文字优化
```css
body {
  line-height: 1.65（从1.6）
  text-rendering: optimizeLegibility
  -moz-osx-font-smoothing: grayscale
}

h1-h6 {
  line-height: 1.3
  letter-spacing: -0.01em
  font-weight: 600
}

p {
  line-height: 1.65
  margin-bottom: 0
}
```

#### ✅ 顶部导航栏优化
```css
背景：纯白色（更清爽）
边框：1px → 2px
标题字号：base → lg（更大）
标题字距：-0.02em（更紧凑）
Eyebrow字距：0.08em（更宽松）
按钮高度：32px → 36px
按钮内边距：增加
按钮背景：添加微妙背景色
按钮hover：上浮效果
```

#### ✅ 侧边栏导航优化
```css
导航项高度：38px → 42px（更舒适）
图标大小：16px → 18px（更清晰）
间距：gap增加
字距：0.005em（微调）
Active状态：
  - 渐变背景
  - 2px边框
  - 阴影效果
Hover：右移4px
Small文字：
  - 透明度：0.7 → 0.75
  - 行高：1.3
  - 间距：2px
```

**效果：**
- ✅ 文字清晰易读
- ✅ 元素间距协调
- ✅ 视觉节奏舒适
- ✅ 交互反馈明确
