## 🔍 已知行为

### 正常行为

1. **首次加载较慢**
   - 原因：需要下载 JS/CSS 资源
   - 后续访问会快很多（浏览器缓存）

2. **WPS 状态显示 OFFLINE**
   - 原因：需要安装和启动 WPS MCP 桥接
   - 不影响其他功能使用

3. **API 状态显示 QUEUE**
   - 原因：默认使用文件队列，不直接调用 API
   - 可在"工作台设置 → API 设置"中配置即时 API

### 不是 Bug 的现象

1. **关系窗口可以与背景交互**
   - 这是**故意的设计**
   - 该窗口不是模态的
   - 测试验证了 `not.toHaveAttribute("aria-modal", "true")`

2. **某些端点未验证**
   - 6/9 端点已保护（核心端点）
   - 剩余 3 个端点风险较低
